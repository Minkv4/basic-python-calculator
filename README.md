# basic-python-calculator
A simple **command-line calculator** built using Python.
The program allows users to perform basic arithmetic operations through a **menu-driven interface**.
This project was created as a beginner exercise to practice core programming concepts such as loops, conditionals, and user input handling.
---
# ✨ Features
* 📋 **Menu-based interface**
* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🔙 Return to main menu
* ❌ Exit program
* ⚠️ Basic error handling (division by zero)

# ▶️ How to Run
